## v1.001 — 初始版本

### 新功能
- 32×8 WS2812 矩阵时钟，显示 HH:MM
- 日期列 / 星期列 / 天气列 像素柱状图
- 通过 OpenWeatherMap 免费 API 获取天气
- NTP 时间同步，每 60 分钟更新一次
- 无 WiFi 时使用手动时间，每小时重试连接
- 恢复模式：长按按钮或崩溃复位时进入
- OTA 固件升级（浏览器界面）

### 已知限制
- 首次使用需通过浏览器配置 API Key 和城市名称
- 需要 OpenWeatherMap 免费账号
